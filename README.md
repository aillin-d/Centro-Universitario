# Sistema de Cadastro de Centro Universitário

Sobre: o aplicativo tem como objetivo o cadastro de diferentes tipos de usuários relacionados a um centro universitário

## Usuários:
Professor \
Aluno \
Fornecedor 

## Funcionalidades:
Cadastrar Usuário \
Consultar Cadastro \
Atualizar Cadastro \
Excluir Cadastro
